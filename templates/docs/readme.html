Docket

Welcome to Docket! To get started, you can run with [python] main.py
To use a custom port, pass the --port argument like this: python main.py --port 8080

Docket automatically shows all .html files stored in /templates/docs/
Default pages: /, /readme
The API to make your own custom index page is at /api/pages. It stores the endpoints for all pages in /templates/docs/.
